# Analysis-on-Students-Performance
This project focuses on predicting student exam performance using key academic factors such as study hours and attendance. The goal is to demonstrate how data analysis and machine learning can be applied to identify patterns that influence student success and support early academic intervention.
In this project, a regression model was built to analyze the relationship between:
Hours Studied
Attendance
Exam Scores
Using a dataset of 6,603 student records, the model was trained to predict exam scores and evaluate how well these factors influence academic performance.
Tools & Technologies
Python
Pandas & NumPy
Scikit-learn
Jupyter Notebook
